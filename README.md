# EcoWave – Join the Green Revolution

## Project Poster

![EcoWave Poster](task%201.png)

### Description
EcoWave is a sustainability awareness campaign that promotes eco-friendly habits such as recycling, conserving water, saving electricity, planting trees, and using sustainable transport. Through impactful visuals and clear messaging, it inspires collective action toward a cleaner, greener, and healthier future.
